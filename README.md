# Трекер заявок

Простое CRUD-приложение с консольным интерфейсом.
Можно добавлять/удалять/изменять заявки, а так же выводить все текущие.

## Технологии
* Java 15
* Travis-CI
* JDBC
* PostgreSQL
* Hibernate
* Junit
* Mockito
* Liquibase
